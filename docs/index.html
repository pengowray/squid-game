Welcome to Squid Game (2007)

The original Squid Game by Pengo Wray, created in Flash.

Instructions

  • Click to attach an arm.
  • Hold left mouse button to release all arms
  • (optional) Arrow keys for free movement. You get a bonus achievement for not using keyboard (or you did on Kongregate anyway).
  • Collect "arm coins" to receive additional tentacle arms.
  • Goal: Collect all the coins to be disappointed that that's all there is. There is no level 2.
  • Play the tutorial for more instructions.

About This Project

This game was originally made in Adobe Flash in 2007. In one development version the balloons actually floated. I don't remember why they stopped doing that. Probably a bug. After releasing this travesty of a game I gave up on writing my own physics engines and started using the open source Box2D engine instead, which simulated physics far less jankily.

Squid Game was made to be played on kongregate.com where it had integrated achievements and a scoreboard, but it's no longer playable there. (archive screenshot)

With the end of Flash Player, it stopped being playable. Thanks to the Ruffle project, I can finally make it "playable" online again. It's still not very playable (it never was), but you can play it. Have fun.
